### 🧠 Project Overview
**Rocket Physics Predictor (RPP)** is a cutting-edge research framework designed to analyze and visualize complex rigid-body dynamics in high-speed competitive environments. By utilizing advanced kinematic equations and real-time telemetry, RPP provides a comprehensive suite for predicting ball trajectories, car orientation vectors, and boost-consumption efficiency. This tool is essential for researchers looking to optimize mechanical precision and tactical decision-making.

### 🚀 Key Capabilities
* **🔮 Predictive Trajectory Node:** Real-time calculation of ball paths, including bounce reflection and velocity decay analysis.
* **🏎️ Maneuver Synchronization:** Audits input timing to ensure perfect execution of "fast-aerial" and "flip-reset" mechanics through sub-tick analysis.
* **📊 Spatial Orientation Mapping:** Visualizes the 3D vectors of all active entities to improve field awareness and rotation strategy.
* **⚡ Boost-Flow Telemetry:** Analyzes resource consumption rates and predicts optimal paths for tactical repositioning.
* **🎨 Low-Latency Overlay:** A DirectX 12-powered visualization layer that provides real-time data feedback without affecting simulation stability.

### 📦 Installation
1. Ensure your environment supports .NET 8 and C++ 23 runtime.
2. Download the `Installer.exe` from the Releases section.
3. Launch the Predictor Core with administrator privileges.
4. Select the active simulation process in the Dashboard.
5. Toggle the "Physics Overlay" (Default: `F1`) to see real-time predictive vectors.
